# Tech Stack

# Backend

> Node.js

> Express.js

> Typescript

> Redis

# Frontend

> React.js

> Typescript

# Instructions to Run This

> git clone https://github.com/blockchain-app/Md-Sazzadul-Coding-Challenge

> open the project in vs code or any editor of your favorite

# Backend

> cd backend

> npm install

> npm run dev

> npm run test

> npm run build - for production

# Frontend

> cd frontend

> npm install

> npm start

> npm run test

> npm run build - for production
